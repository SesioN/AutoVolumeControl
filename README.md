# AutoVolumeControl

AutoVolumeControl - A tool that helps you control volume for the digital output on creative devices
Hello,

if you are having the same issues with your creative product as I was, where you want to adjust the volume of your music etc. via the master windows volume bar, your keyboard or directly on the creative device and it doesn't work for the digital output then here comes help.

(https://www.reddit.com/r/SoundBlasterOfficial/comments/h9oq5s/update_and_status_thread_06142020/fwwcprf?utm_source=share&utm_medium=web2x)

I've written a small windows application that will appear on your system tray bar.It will adjust the volume of all running applications accordingly to the main volume bar on the default playback device. (In my case "Speakers Sound BlasterX G6")

Requirements:

Windows 10

Download:

https://drive.google.com/file/d/1hc9pbB5U1pXH9vxAgqWsQoj2gzmUIl48/view?usp=sharing

Just start it and put to windows auto start.
